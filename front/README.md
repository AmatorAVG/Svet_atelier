# YouTube Summary

Сервис для получения кратких изложений YouTube видео с помощью ИИ.

## О проекте

YouTube Summary автоматически получает транскрипт YouTube видео и генерирует краткое изложение на русском языке с помощью LLM.

### Возможности
- 📝 Получение транскрипта YouTube видео (до 15 минут)
- 🤖 Генерация краткого изложения с помощью ИИ
- 🌐 Поддержка русского языка
- ⚡ Быстрая обработка
- 🎨 Современный минималистичный дизайн

### Технологии
- **Framework**: Next.js 16.2.0 (App Router)
- **Language**: TypeScript
- **UI**: shadcn/ui + Tailwind CSS
- **AI SDK**: Vercel AI SDK
- **API**: Supadata.ai (транскрипты) + OpenRouter (LLM)

## Быстрый старт

### Предварительные требования
- Node.js 18+ 
- pnpm (рекомендуется) или npm
- API ключи:
  - [Supadata.ai](https://supadata.ai) - для получения транскриптов
  - [OpenRouter](https://openrouter.ai) - для генерации summary

### Установка

1. **Клонировать репозиторий**
   ```bash
   git clone <your-repo-url>
   cd front
   ```

2. **Установить зависимости**
   ```bash
   pnpm install
   # или
   npm install
   ```

3. **Настроить переменные окружения**
   
   Откройте файл `.env.local` и замените значения на ваши API ключи:
   
   ```bash
   SUPADATA_API_KEY=ваш_ключ_supadata
   OPENROUTER_API_KEY=ваш_ключ_openrouter
   ```
   
   > 💡 Файл `.env.example` содержит шаблон с комментариями

4. **Запустить dev сервер**
   ```bash
   pnpm dev
   # или
   npm run dev
   ```

5. **Открыть приложение**
   
   Перейдите по адресу: [http://localhost:3000](http://localhost:3000)

## Получение API ключей

### Supadata.ai
1. Зарегистрируйтесь на [supadata.ai](https://supadata.ai)
2. Перейдите в Dashboard → API Keys
3. Создайте новый API ключ
4. Скопируйте ключ в `.env.local`

### OpenRouter
1. Зарегистрируйтесь на [openrouter.ai](https://openrouter.ai)
2. Перейдите в Settings → API Keys
3. Создайте новый API ключ
4. Скопируйте ключ в `.env.local`

> 💡 Модель `minimax/minimax-m2.5:free` бесплатная, но может потребоваться пополнить баланс для других моделей.

## Использование

1. Откройте приложение в браузере
2. Вставьте ссылку на YouTube видео в поле ввода
3. Нажмите кнопку "Краткое содержание"
4. Дождитесь обработки (обычно 10-30 секунд)
5. Получите краткое изложение видео

### Ограничения
- Видео должно быть не длиннее 15 минут
- Видео должно иметь субтитры (автоматические или ручные)
- Поддерживается только русский язык для summary

## API

### POST /api/summarize

Получает краткое изложение YouTube видео.

**Request:**
```json
{
  "url": "https://www.youtube.com/watch?v=VIDEO_ID"
}
```

**Response (Success):**
```json
{
  "title": "Название видео",
  "summary": "Краткое изложение видео..."
}
```

**Response (Error):**
```json
{
  "error": "Описание ошибки"
}
```

**Коды ошибок:**
- `400` - Некорректный URL
- `500` - Ошибка сервера (нет субтитров, видео слишком длинное, etc.)

## Troubleshooting

### Ошибка "SUPADATA_API_KEY не настроен"
- Убедитесь, что файл `.env.local` существует в корне проекта
- Проверьте, что ключи записаны правильно
- Перезапустите dev сервер после изменения `.env.local`

### Ошибка "Субтитры для этого видео недоступны"
- Видео должно иметь субтитры (автоматические или ручные)
- Попробуйте другое видео с субтитрами

### Ошибка "Видео слишком длинное"
- Поддерживаются видео до 15 минут
- Попробуйте более короткое видео

### Медленная обработка
- Обычно занимает 10-30 секунд
- Зависит от длины видео и скорости API
- При проблемах проверьте статус Supadata.ai и OpenRouter
